# SUBAD

**Virtual rooms. Meet, decorate, connect, explore.**

Subad is a sandboxed social space where every user gets a starting room. Personalize it with posters, mirrors, windows, music, GIFs, portals — and when you're ready, connect with others.

## Philosophy

- **Simple** — no frameworks, no databases, no cloud
- **Creative** — rooms are canvases, not profiles
- **Unorthodox** — breaks the social media mold
- **Secure** — sandboxed by default, crypto-optional
- **Lightweight** — runs in a browser, no install
